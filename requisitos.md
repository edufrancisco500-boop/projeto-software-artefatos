# Requisitos do Projeto de Software

## Requisitos Funcionais
1. O sistema deve permitir o cadastro de usuários.
2. O sistema deve permitir alterar dados cadastrais.
3. O sistema deve gerar relatórios simples.
4. O sistema deve salvar dados localmente ou em arquivo.

## Requisitos Não Funcionais
1. A interface deve ser simples.
2. O sistema deve ser fácil de usar.
3. O código deve ser organizado e versionado no GitHub.
